# Sweet Carmesi 🍓

Sitio web de mermeladas artesanales premium.
Proyecto de tienda online con carrito de compra y diseño elegante.

## 🌐 Demo
Puedes ver el sitio aquí:
https://gamersgabriel78-beep.github.io/sweet-carmesi/

## ✨ Características
- Diseño moderno y responsive
- Carrito de compras interactivo
- Control de cantidades
- Estilo visual elegante
- Productos gourmet

## 🛠 Tecnologías usadas
- HTML5
- CSS3
- JavaScript

## 📦 Productos
- Mermelada con Maní
- Mermelada con Ron

## 🚀 Cómo usar el proyecto
1. Clona el repositorio
2. Abre `index.html` en tu navegador
